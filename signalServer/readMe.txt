//Setup

> 1. Make sure to install NodeJS on your server.
     http://nodejs.org/

> 2. Once installed open command line or equivalent and enter.
     npm install socket.io

> 3. Open command and run server.js script.
     node server.js

> 4. Socket IO will start running on port 8080. Make sure to open 
     port to allow connections.

> 5. Open sample WebRTC HTML file and where new Signaling instance is
     created replace holder string with:
     http://yourDomain:8080
